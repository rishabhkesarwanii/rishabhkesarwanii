Stock Tracker

Django Celery
Channel redis

I recently completed a project on a Stock Tracker Django app which incorporated Django channels, Celery, and Redis. The main objective of the project was to allow users to select their desired stock and then view all the relevant details and live updates related to that stock. To achieve this, I used Django channels to enable real-time updates and Celery with Redis to implement a background cronjob for fetching stock data.

  • Automatic fetching of all relevant stock details, including current value, fluctuations, and other important metrics
  • Real-time updates provided by a background cronjob that fetches and updates stock data periodically
  • Implementation of Django channels for WebSocket-based communication between the client and server

With this app, users can easily search for and select any stock they wish to track, and the app will automatically fetch and display all the relevant details of that stock, including its current value, fluctuations, and other important metrics.

Automated Website Deployment

Github Jenkins
Docker AWS ECR

My recent project involved creating an automated pipeline that enables the dockerization of a project and pushing the docker image to AWS ECR. The pipeline is initiated whenever a new commit is made to the project.

  • Utilizes Jenkins pipeline to automate the build and deployment process
  • Detects new commits and automatically initiates the pipeline
  • Dockerizes the entire project and creates a docker image
  • Pushes the docker image to AWS ECR
  • Provides an efficient and scalable way to build, test and deploy projects

Jenkins is used to automate the build and deployment process, which ensures that the pipeline is initiated automatically whenever a new commit is detected. The pipeline then proceeds to dockerize the project, which involves creating a container image with all the required dependencies and libraries. The docker image is then pushed to AWS ECR, which enables easy deployment of the project to the cloud.
